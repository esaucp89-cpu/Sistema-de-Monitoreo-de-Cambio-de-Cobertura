# Sistema-de-Monitoreo-de-Cambio-de-Cobertura
Desarrollo un producto geoespacial semiautomático para la detección y monitoreo de cambios en cobertura forestal dentro de fincas incorporadas al Programa de Pago por Servicios Ambientales, utilizando imágenes Sentinel-2 y tecnologías GIS para apoyar los procesos de seguimiento, control y verificación del programa.
